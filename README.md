## Install

- Node.js 18+ is recommended.

### NPM

To install via NPM:

```
npm install @did-coop/wallet-attached-storage
```

### Development

To install locally (for development):

```
git clone https://github.com/did-coop/wallet-attached-storage.git
cd wallet-attached-storage
npm install
```
